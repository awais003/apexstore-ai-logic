# ApexStore AI: The Agentic RAG Layer

This is the "Brain" of the ApexStore AI ecosystem. It handles semantic retrieval and intelligent processing using the **Atomic Coding Method**.

## 🧠 Key Features

- **Hybrid Search Orchestrator:** Combines keyword matching with vector embeddings.
- **RAG Integration:** Uses Google Gemini (Flash 2.5) for context-aware responses.
- **Vector Store:** Configured for Qdrant/ChromaDB for high-speed retrieval.

## 🛠 Tech Stack

- **Language:** Python 3.x
- **AI Models:** Gemini 2.5 Flash, Hugging Face Embeddings
- **Task Queue:** Celery + Redis
